Mi segundo proyecto!

Shows & Resto

Este fue un MVP React para una plataforma de turismo y gastronomia

    Tecnologías:

  • HTML
  • CSS
  • JavaScript
  • React.js
  • React-Router
  • React Dom
  • Axios framework
  • Google Maps API
  • MaterialUI Library

En este proyecto se realizo un MVP el cual cuenta con gestion de usuarios, un carrousel para mostrar restaurantes y uno para mostrar shows. Se realizo un filtro para la busqueda de restaurantes y shows para facilitar al usuario la experiencia. Cada restaurante cuenta con una pagina en la cual se muestra su carta, su informacion y su ubicacion en google maps.
